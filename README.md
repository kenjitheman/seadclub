## SEAD Club's website

## Installation

```shell
git clone https://github.com/seadclub/seadclub.online
```

- Install all dependencies

```shell
yarn install
```

## Usage

- Run

```shell
yarn start
```

- Run in dev mode

```shell
yarn dev
```
